# Sync Blogs Docs

This folder tracks product strategy, research, and execution for the private AI-assisted blogging platform.

## Current direction

- Core product promise: start with ideas (messy thoughts), not a blank page.
- Optional transcription is supported, but not the core brand position.
- Core differentiation: private writing + personal voice fidelity + post-publish freshness intelligence.

## Suggested operating cadence

- Weekly: competitor changes + model/platform changes review.
- Weekly: stale-content queue triage and publish approvals.
- Biweekly: ship one user-visible improvement to drafting, feedback, or freshness.
